En app.js se agrego las rutas de paciente y turno.
En server.js se agrego las rutas de paciente y turno.
En rutas de turno se agrego las funciones crud.
En modelos se configuro para que se ingrese lo que se pide en el enunciado.
En entidad se construyo el constructor teniendo un medico asignado como esta en el enunciado, donde reutiliza la entidad
de identificador para darle un id a cada turno.
En las funciones esta todo lo relacionado a poder crear o dar un mensaje de error correspondiente para las funciones
de crud.

Al iniciar el proyecto se realizo:
npm init
npm install express
express --view=ejs
npm install
npm run start
